## Ferramentas incluidas neste setup

### GSD — Get Shit Done
- **Repo:** https://github.com/gsd-build/get-shit-done
- **Criador:** TACHES / gsd-build
- **Licenca:** MIT
- **Stars:** 53k+
- O sistema de workflow spec-driven que previne context rot no Claude Code.
  Responsavel pelos hooks de contexto, execucao em ondas paralelas e estrutura de fases.

### SuperClaude Framework
- **Repo:** https://github.com/SuperClaude-Org/SuperClaude_Framework
- **Criador:** SuperClaude-Org
- **Licenca:** MIT
- **Stars:** 20k+
- Framework que adiciona 31 comandos `/sc:` e 20 agentes especializados ao Claude Code.

### repomix
- **Repo:** https://github.com/yamadashy/repomix
- **Criador:** yamadashy
- **Licenca:** MIT
- **Stars:** 20k+
- Empacota toda a codebase num arquivo otimizado para IA, reduzindo tokens drasticamente.

### ccusage
- **Repo:** https://github.com/ryoppippi/ccusage
- **Criador:** ryoppippi
- **Licenca:** MIT
- Dashboard CLI para monitorar consumo de tokens e custos do Claude Code.

### task-master-ai
- **Repo:** https://github.com/eyaltoledano/claude-task-master
- **Criador:** eyaltoledano
- **Licenca:** MIT
- **Stars:** 20k+
- Sistema de gerenciamento de tarefas integrado ao Claude Code com suporte a PRDs.

### claude-squad
- **Repo:** https://github.com/smtg-ai/claude-squad
- **Criador:** smtg-ai
- **Licenca:** MIT
- Terminal app para orquestrar multiplos agentes Claude Code em paralelo via tmux + git worktrees.

## MCP Servers

### Sequential Thinking MCP
- **Pacote:** `@modelcontextprotocol/server-sequential-thinking`
- **Repo:** https://github.com/modelcontextprotocol/servers
- **Criador:** Anthropic / Model Context Protocol
- Quebra tarefas complexas em passos logicos antes de executar.

### Context7 MCP
- **Pacote:** `@upstash/context7-mcp`
- **Repo:** https://github.com/upstash/context7
- **Criador:** Upstash
- Busca documentacao atualizada diretamente dos repositorios oficiais em tempo real.

### Filesystem MCP
- **Pacote:** `@modelcontextprotocol/server-filesystem`
- **Repo:** https://github.com/modelcontextprotocol/servers
- **Criador:** Anthropic / Model Context Protocol
- Acesso seguro a arquivos locais com fronteiras configuradas.
